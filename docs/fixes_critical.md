2. `BattleIntroductionEvent.isFinished` is set inside `render()` ‚Äî same contract violation; **fix**: move `isFinished = true` into `update()` with a timer or a condition based on the intro animation completing

4. `ExpAnimationEvent` caps exp at `expNextLevel` and resets to 0 on level-up (`ExpAnimationEvent.java:17-19`) ‚Äî overflow exp is silently discarded; a Pokemon that earns enough exp for two levels only levels once; **fix**: after leveling up, compute `overflowExp = newExp - expNextLevel`; recalculate `expNextLevel` for the new level; if `overflowExp` still exceeds the new `expNextLevel`, loop and level again; set `currentExp` to the final remainder
